Instalación

El primer paso es instalar Rust. Descargaremos Rust a través de rustup, un herramienta de línea de comandos para administrar las versiones de Rust y las herramientas asociadas. Necesitarás una conexión a internet para la descarga.

Nota: Si prefieres no usar rustup por alguna razón, consulta la La página Otros métodos de instalación de Rust para obtener más opciones

.

Los pasos siguientes instalan la última versión estable del compilador de Rust. Las garantías de estabilidad de Rust garantizan que todos los ejemplos del libro que compile seguirá compilándose con las versiones más recientes de Rust. La salida podría difieren ligeramente entre las versiones porque Rust a menudo mejora los mensajes de error y advertencias. En otras palabras, cualquier versión nueva y estable de Rust que instales usando estos pasos deberían funcionar según lo esperado con el contenido de este libro.

Notación de línea de comandos

En este capítulo y a lo largo del libro, mostraremos algunos de los comandos que se utilizan en terminal. Todas las líneas que debes introducir en una terminal comienzan por $. Tú no necesitas escribir el carácter $; es la línea de comandos que se muestra a indica el inicio de cada comando. Por lo general, las líneas que no comienzan por $ muestran el resultado del comando anterior. Además, es específico de PowerShell

los ejemplos usarán > en lugar de $.

Instalación de rustup en Linux o macOS

Si utilizas Linux o macOS, abre una terminal e introduce el siguiente comando:

$ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh

El comando descarga un script e inicia la instalación del rustup herramienta, que instala la última versión estable de Rust. Es posible que se le pregunte para tu contraseña. Si la instalación se realiza correctamente, aparecerá la siguiente línea:

Rust is installed now. Great!

También necesitarás un enlazador, que es un programa que Rust usa para unir sus salidas compiladas en un solo archivo. Es probable que ya tengas uno. Si obtienes errores de vinculación, debe instalar un compilador de C, que normalmente incluirá un enlazador. Un compilador de C también es útil porque algunos paquetes comunes de Rust dependen de Codifica en C y necesitará un compilador en C.

En macOS, puedes obtener un compilador de C ejecutando:

$ xcode-select --install

Los usuarios de Linux generalmente deberían instalar GCC o Clang, según sus documentación de la distribución. Por ejemplo, si usas Ubuntu, puedes instalar el paquete build-essential

.

Instalación de rustup en Windows

En Windows, vaya a https://www.rust-lang.org/tools/install y siga las instrucciones para instalar Rust. En algún momento de la instalación, podrás recibirás un mensaje en el que se te explica que también necesitarás las herramientas de compilación de MSVC para Visual Studio 2013 o posterior

.

Para adquirir las herramientas de compilación, necesitará instalar Visual Studio 2022. Cuando se le pregunte qué cargas de trabajo instalar, incluya:

  • «Desarrollo de escritorio con C++»
  • El SDK de Windows 10 u 11
  • El componente del paquete de idioma inglés, junto con cualquier otro paquete de idioma de tu elección

El resto de este libro usa comandos que funcionan tanto en cmd.exe como en PowerShell. Si hay diferencias específicas, explicaremos cuáles usar

.

Solución de problemas

Para comprobar si tienes Rust instalado correctamente, abre un shell e introduce esto línea:

$ rustc --version

Deberías ver el número de versión, el hash de confirmación y la fecha de confirmación de la última versión estable que se ha publicado, en el siguiente formato:

rustc x.y.z (abcabcabc yyyy-mm-dd)

Si ve esta información, ¡ha instalado Rust correctamente! Si no lo haces consulta esta información, comprueba que Rust esté en tu variable de sistema %PATH% como sigue.

En Windows CMD, utilice:

> echo %PATH%

En PowerShell, utilice:

> echo $env:Path

En Linux y macOS, usa:

$ echo $PATH

Si todo es correcto y Rust sigue sin funcionar, hay una serie de lugares donde puede obtener ayuda. Descubra cómo ponerse en contacto con otros rustáceos (a) apodo tonto (nos llamamos a nosotros mismos) en la página de la comunidad

.

Actualización y desinstalación

Una vez que Rust se instala a través de rustup, la actualización a una versión recién lanzada es fácil. Desde tu shell, ejecuta el siguiente script de actualización:

$ rustup update

Para desinstalar Rust y rustup, ejecuta el siguiente script de desinstalación desde tu

shell: 
$ rustup self uninstall

Documentación local

La instalación de Rust también incluye una copia local de la documentación, por lo que que puedes leerla sin conexión. Ejecute rustup doc para abrir la documentación local en tu navegador.
